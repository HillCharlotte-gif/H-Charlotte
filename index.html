Glossary (Unit 1 & Unit 2)

Lossless Compression
A data compression algorithm that allows the original data to be perfectly reconstructed from the compressed data.
Image
A type of data used for graphics or pictures.
RGB
The color model that uses varying intensities of red, green, and blue light, added together to reproduce a broad array of colors.
Metadata
Data that describes other data. For example, a digital image may include data that describe the size of the image, number of colors, or resolution.
Lossy Compression
A data compression method that uses inexact approximations, discarding some data to represent the content.
Pixel
The fundamental unit of a digital image, typically a tiny square or dot which contains a single point of color of a larger image.
ASCII
The universally recognized raw text format that any computer can understand.
Binary Numbers
Numbers expressed in the base-2 numeral system using only 0 and 1.
Convert from Binary to Decimal / Decimal to Binary
Processes for switching between base-2 (binary) and base-10 (decimal).
2ⁿ Formula
The number of bits, n, encodes 2ⁿ unique possible values.
Compression Rate
Calculated as (# bytes compressed out ÷ # bytes in original message).
TCP (Transmission Control Protocol)
Provides reliable, ordered, and error-checked delivery of packets on the internet.
DNS (Domain Name System)
The service that translates URLs into IP addresses.
Packets
Small chunks of information that have been carefully formed from larger chunks of information.
HTTP (Hypertext Transfer Protocol)
The protocol used for transmitting web pages over the Internet.
IP (Internet Protocol)
A number assigned to any item connected to the Internet.
HTML
The language in which the content and formatting of a web page are written.
Bandwidth
Transmission capacity measured by bit rate.
URL
An easy-to-remember address for calling a web page.
Computer Network
A group of interconnected computing devices capable of sending or receiving data.
Router
A type of computer that forwards data across a network.
Fault Tolerant
A system that can continue to function even if individual components fail.
Scalability
The capacity for a system to change in size and scale to meet new demands.
